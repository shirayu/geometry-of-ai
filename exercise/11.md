# 第11回：感覚の統合

## 問題 11-1 ★ CLIPのコントラスト学習

CLIPのコントラスト学習で、正例ペア（テキストと対応する画像）を近づけ、負例ペア（テキストと無関係な画像）を遠ざける操作を、球面上の「引力・斥力」として説明せよ。

**ヒント:** 損失関数の勾配の向きを考えよ。

**解答:**
**コントラスト損失（InfoNCE）の概形:**
$L = -\log( \exp(\text{sim}(t,i)/\tau) / \Sigma_j \exp(\text{sim}(t,i_j)/\tau) )$

**勾配の効果:**

- 正例ペア $(t, i)$ ： $sim(t, i)$ を大きくする方向に勾配 → 近づける（引力）
- 負例ペア $(t, i_j)$ ： $sim(t, i_j)$ を小さくする方向に勾配 → 遠ざける（斥力）

**球面上での解釈:**

- 正規化された表現は単位球面上に存在
- 引力：正例ペアの角度を小さくする（同じ方向に向ける）
- 斥力：負例ペアの角度を大きくする（直交または反対方向に向ける）

**結果:** 学習後、意味的に関連するテキストと画像は球面上で近い位置に、無関係なものは遠い位置に配置される。

## 問題 11-2 ★★ マルチモーダル統合の戦略

マルチモーダル学習で「統一多様体」を目指すアプローチと「多様体間の翻訳」を目指すアプローチの、それぞれの利点と欠点を挙げよ。

**ヒント:** 情報の損失、計算コスト、汎用性を考えよ。

**解答:**
**統一多様体アプローチ（CLIP型）:**

| 利点 | 欠点 |
| --- | --- |
| シンプル：単一の埋め込み空間 | モダリティ固有の情報が失われうる |
| 直接比較可能：テキストと画像の類似度が直接計算可能 | 「無理やり同じ空間」の歪み |
| 汎用的：新しいモダリティも同じ空間に追加可能（理論上） | 空間の容量に限界 |

**多様体間翻訳アプローチ:**

| 利点 | 欠点 |
| --- | --- |
| モダリティ固有の構造を保存 | 翻訳関数の学習が必要 |
| より表現力が高い可能性 | 計算コストが高い |
| 可逆性を持たせやすい | 新モダリティ追加時に翻訳関数も追加必要 |

**現実の傾向:** 実用的にはCLIP型の統一アプローチが主流だが、高精度が必要なタスクでは翻訳アプローチも使われる。
